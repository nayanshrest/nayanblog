Antivirus

Anti Virus is a computer program that is used to scan,detect and remove viruses from the computer system.It is a software tool that scans for viruses and takes action to remove them.
An anti-virus software is used to prevent and protect our computer from viruses.

Some popular anti-virus softwares:-

  1. Norton anti-virus(NAV)
  2. AVG(Anti Virus Guard)
  3. Kaspersky Anti Virus(KAV)
  4. ESET Nod32 Anti-virus
  5. Trend Micro Antivirus

Protection from Computer Viruses

Computer virus causes great harm in the computer system in different ways. THey irritate the user while using and working in the computer. Every year millions and billions of dollars is
being invested due to computer virus. Therefore, we should follow the preventive measures to avoid virus infection.

Preventive measures from computer viruses:-

  1. Installing latest antivirus on the computer and updating it regularly.
  2. Scanning the computer and external storage devises for viruses on regular basis.
  3. Locking the computer using strong password.
  4. Using the genuine(not pirated) software.
  5. Not allowing other people to use the computer without permission.
  6. Check the new software for virus before installing.
