Computer virus


Computer virus is a man made program which destroys the software of the computer.It is a
small computer program that harms computer software and data in many ways.Since, the computer
virus is malicious computer program, it disrupts the normal functioning of the computer.
There are many computer viruses like -
1)I Love you 2)Time Bomb 3)Aids 4) C-Brain 5) Logic Bomb

Some symptoms of computer virus are :-
1)Viruses reduce the memory space of the computer.
2)Viruses change the size of computer files.
3)Viruses change the entension of files.
4)Viruses show unusual error messages on the computer scee.
5)Viruses may hang or shutdown the computer automatically, while working

Some type's of viruses are as follows:-
1)Message carrying viruses: The message carrying virus is a type of computer virus which does not infect the program and files but entertains
the user by producing and displaying unnecessary message's, quotation, pictures,etc on the screen.The computer user may disturb due to such messages.

2)Boot sector virus: The boot sector is a type of computer virus which infects the information system during the start up process.
It tends to create a bad sector of the hard drive or the floppy disc.

3) System infector virus:The system infector is a type of computer virus that infects the various parts of operating system or Master Control Program (MCP).
It also damages the system files of operating system which alters the computer while booting.

4) Multi partite virus: The multi partite is a type of computer virus that infects both executable files and boot sector.
It is a hybrid virus which is comparaatively more infectious and difficult to remove from the computer.

5) Polymorphic virus: The polymorphic is one of the most dangerous and harmful computer viruses.This virus cannot be easily detected and removed
by antivirus program.
